CoreImagePlayground
===========

## What is this

Sample App of

- `SimpleCamera`
- `SimpleAVPlayer`
- `GPUCIImageView`
- `CIFilterExtension`


## Build

```sh
brew install swiftlint

brew install carthage

carthage bootstrap --no-use-binaries --no-build --use-submodules --platform iOS
```
